### ğŸ—ï¸ 1. **Creational Patterns**

| Pattern Name         | Use Case                                       |

### ğŸ§± 2. **Structural Patterns**

> Concerned with **object composition and relationships**

| Pattern Name  | Use Case                                          |

### ğŸ¤ 3. **Behavioral Patterns**
